Installer Xilinx Vivado

  • Télécharger Vivado
  • Installer Vivado
  • Démarrer Vivado

Si vous souhaitez travailler sur votre propre ordinateur, vous pouvez installer les outils de développement en suivant la procédure ci-dessous.

Xilinx effectue régulièrement des modifications de son site web et de l’interface utilisateur de ses outils. Si vous remarquez des erreurs dans cette page, merci de nous les signaler.

En tronc commun de la formation ingénieur sous statut étudiant, nous travaillons avec la suite Xilinx Vivado en version 2019.1 sous Ubuntu 20.04.

Certaines formations dispensées à l’ESEO utilisent également Vivado sous Windows. Renseignez-vous auprès de vos enseignants pour savoir quelle version installer.

L’installation de ces outils nécessite le téléchargement de fichiers très volumineux. Nous vous déconseillons de le faire en utilisant le réseau Wi-Fi de l’école.

Télécharger Vivado

  1. Accédez à la page des téléchargements sur le site de Xilinx.
  2. Descendez dans la page jusqu’à « Vivado Design Suite - HLx Editions - 2019.1 Full Product Installation ».
  3. Suivez le lien correspondant à votre choix :
    • L’installeur pour Linux : Vivado HLx 2019.1: WebPACK and Editions - Linux Self Extracting Web Installer
    • L’installeur pour Windows : Vivado HLx 2019.1: WebPACK and Editions - Windows Self Extracting Web Installer
  4. Avant le téléchargement, vous devez créer un compte utilisateur chez Xilinx.

Installer Vivado

Si vous installez Vivado pour Linux, rendez le fichier Xilinx_Vivado_SDK_Web_2019.1_0524_1430_Lin64.bin exécutable et exécutez-le. Si nécessaire, remplacez le nom du dossier Téléchargements par le chemin du dossier où se trouve le fichier que vous avez téléchargé :

cd ~/Téléchargements
chmod +x Xilinx_Vivado_SDK_Web_2019.1_0524_1430_Lin64.bin
./Xilinx_Vivado_SDK_Web_2019.1_0524_1430_Lin64.bin

Si vous installez Vivado pour Windows, exécutez le fichier Xilinx_Vivado_SDK_Web_2019.1_0524_1430_Win64.exe que vous venez de télécharger.

Suivez les instructions en respectant les choix suivants :

  1. À l’étape « Select Edition to Install », choisissez « Vivado HL WebPACK ».
  2. Choisissez les éléments à installer. Dans le cadre de cette activité, les options suivantes sont suffisantes :
    • Design Tools → Vivado Design Suite
    • Devices → Production Devices → 7 series → Artix 7
    • Installation Options → Install Cable Drivers
  3. Vérifiez ou modifiez l’emplacement où se fera l’installation :
    • Sous Linux : /opt/Xilinx
    • Sous Windows : C:\Xilinx

Avec ces options, l’installation nécessite environ 23 Go d’espace libre sur votre disque dur.

Sous Linux, l’installation des pilotes de sondes JTAG (Cable Drivers) n’est pas toujours possible à partir de l’installeur. Pour compléter l’installation, exécutez ces commandes :

cd /opt/Xilinx/Vivado/2019.1/data/xicom/cable_drivers/lin64/install_script/install_drivers/
sudo ./install_drivers

Démarrer Vivado

Sous Linux, ouvrez un terminal et démarrez Vivado à l’aide des commandes suivantes :

source /opt/Xilinx/Vivado/2019.1/settings64.sh
export LANG=en_US.UTF-8
export LC_ALL=en_US.UTF-8
vivado

Les affectations LANG=... et LC_ALL=... permettent de contourner un bug de Vivado 2019.1 si votre environnement de travail est configuré en français.

Sous Windows, Vivado se trouve dans le menu des applications, dans la rubrique « Xilinx Design Tools ».
